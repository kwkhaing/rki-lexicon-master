# Contributing to Rakhine Lexicon

We welcome contributions from linguists, native speakers, and language enthusiasts.

## How to Contribute
1. Fork the repository
2. Add new entries to `data/processed/new_entries.tsv`
3. Follow the data format guidelines
4. Submit a pull request

## Data Format Requirements
- Use Myanmar Unicode for Rakhine words
- Follow ISO 15919 for romanization
- Include IPA transcriptions
- Provide example sentences where possible
- Add audio recordings if available

## Validation
All entries are validated for:
- Unicode correctness
- IPA format compliance
- Required field completion
- Audio quality (if provided)

## Types of Contributions
### For Linguists:
- Add entries with full linguistic annotation
- Provide IPA transcriptions
- Include example sentences
- Add etymological information

### For Native Speakers:
- Record pronunciations
- Validate translations
- Provide dialect variations
- Suggest example sentences

### For Developers:
- Build tools on top of the API
- Create visualizations
- Develop learning apps
- Improve processing scripts
